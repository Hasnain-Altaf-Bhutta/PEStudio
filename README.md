# PEStudio

[PEStudio](https://pestudio.org/) is a powerful Windows executable (PE) analysis tool used to analyze suspicious files, malware, and software for security research and reverse engineering.  

## Features

- Analyze PE files without executing them  
- Detect hidden threats and suspicious behavior  
- Lightweight and portable – no installation required  
- Provides detailed report on executable characteristics
